Este archivo esta escrito segun las normas
de presentacion del Proyecto Final de la
carrera Ingeniería Electromecánica que
pertenece a la Universidad Tecnológica
Nacional Facultad Regional Reconquista.

Para tener un pantallaso de como queda 
finalmente abri el archivo "main.pdf"

Para comenzar inicia el archivo "main.tex"
en este se incluyen los demas archivos
comenzando por la clase de documento, los
paquetes utilizados, las configuraciones de
encabezado y pie de pagina, las configuraciones
generales, indices, capitulos y bibliografia

Espero sea de utlilidad, cualquier mejora será 
bienvenida.
